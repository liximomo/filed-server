# file download server
serve your local files that can be a browser downloads.

## Installation
```
  npm install --global filed-server 
```

## Usage

```
  filed-server [flags]
```

## Flags

| Flag      | Short Flag  | Description      |
| ----------|:-----------:| ----------------:|
| --port    | -p          |  port to listen  |
| --host    | -h          |  server address  |
